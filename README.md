# 🛵 FreshRush

### 🚀 Freshness Delivered Fast

FreshRush is a local delivery platform designed to connect users with nearby shops including restaurants, street food vendors, grocery stores, and more. Users can browse shops, explore products, and place orders easily.

---

## 📌 Features

### 🏠 User Features
- 📍 Location-based shop discovery
- 🍔 Restaurants & Street Food listing
- 🛒 Add to Cart & Checkout system
- 🚚 Delivery & Self Pickup option
- 💳 Cash & Online Payment (UPI)
- 📞 Call shop directly
- 💬 WhatsApp order support

---

### 🏪 Shop Owner Features
- 🧾 Manage shop profile
- 📦 Add/Edit products
- 📥 Accept/Reject orders
- 📊 Track sales

---

### 🛠️ Admin Features
- ➕ Add / Remove shops
- 📦 Manage orders
- 👨‍💼 Assign deliveries
- 📈 Monitor platform activity

---

## 📱 App Flow

1. User opens app  
2. Selects location  
3. Browses shops  
4. Views products  
5. Adds items to cart  
6. Chooses delivery or self pickup  
7. Completes checkout  
8. Order placed successfully 🎉
